[COMMONS-1047] It's not possible to have two BocBooleanValues with distinct ResourceSets on the same page.

Component/s: Web
Issue Type: Bug
Resolution: Fixed
Status: Closed


Since the RegisterStartupScript call which calls the "BocBooleanValue_InitializeGlobals(...)" function in the OnPreRender of BocBooleanValue gets always registered with the same (only type dependant) key, multiple ResourceSets can't be registered on one Page.
