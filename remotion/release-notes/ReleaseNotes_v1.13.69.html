[RM-3040] SecMan: broken validation of parent hierarchy in EditGroupControl

Component/s: SecurityManager
Issue Type: Bug
Resolution: Fixed
Status: Closed


Solution:
EditGroupControl.ascx.cs:

private bool IsParentHierarchyValid (Group group)
{
  var groups = group.CreateSequence (g => g.Parent, g => g != null && g.Parent != group);
  if (groups.Any() && groups.Last().Parent != null)
    return false;
  return true;
}

[RM-3051] Semantic of UrlUtility.GetAbsoluteUrl has changed for non-rooted URLs

Component/s: Web
Issue Type: Breaking Change
Resolution: Fixed
Status: Closed


Previously, non-rooted paths were resolved by pretending the application root. This implicit assumption is not guaranteed to be always valid. Therefor, the new semantic is to throw an ArgumentException if a non-rooted path is passed into the method.

[RM-2957] Provide a GetValueOrDefault Extension-Method for IDictionary

Component/s: Core
Issue Type: New Feature
Resolution: Fixed
Status: Closed


Located in Remotion/Collections.
Extension-Class: DictionaryExtensions
Signature: TValue GetValueOrDefault<TKey, TValue> (this IDictionary<TKey, TValue> dictionary, TKey key)

Accesses the dictionary via TryGetValue, if no value is found, the default value for TValue is returned.
