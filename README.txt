Note: localhost is assumed for MSSQL

1) Create Database Phonebook in MSSQL 
2) Execute SetupDB.sql
3) Compile Solution
4) Set PhoneBook.Sample as Start-Project (used to fill some values in db)
5) Start
6) Set PhoneBook.Web as Start-Project 
7) Start
