[RM-2612] SQL Backend: Add support for simple custom method call transformer registration with new backend

Component/s: Data.Linq for re-store
Issue Type: New Feature
Resolution: Fixed
Status: Closed


Implementations of the new ILinqParserCustomizers interface can be registered via IoC. When the first LINQ query is executed, the QueryFactory retrieves these implementations and uses them to register custom method call transformers, result operator handlers, and node type parsers. If a customizer registers a transformer, handler, or parser for an item that already was registered, the customizer overrides the default implementation.

[RM-1058] SQL Backend: Support for String.Length property

Component/s: Data.Linq
Issue Type: New Feature
Resolution: Fixed
Status: Closed


Support for queries of the following form: from c in Companies where c.Name.Length > 7 select c.
