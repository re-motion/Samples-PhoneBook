[COMMONS-1690] Add StopwatchScope.Checkpoint method to enable incremental time measurements

Component/s: Core
Issue Type: New Feature
Resolution: Fixed
Status: Closed


using (var scope = StopwatchScope.Create(s_log, "Performing work: {elapsed} ({context})"))
{
  Thread.Sleep (1000);
  scope.Checkpoint ("step 1");
  Thread.Sleep (1000);
  scope.Checkpoint ("step 2");
  Thread.Sleep (1000);
}

This results in roughly the following output:

{pre}
Performing work: 00:00:01 (step 1)
Performing work: 00:00:02 (step 2)
Performing work: 00:00:03 (end){pre}

The following placeholders are available:

  • context - the context given to the Checkpoint method, "end" for the end of the scope
  • elapsed - the total elapsed time in the standard TimeSpan format ([-][d.]hh:mm:ss[.fffffff])
  • elapsed:ms - the total elapsed time in milliseconds
  • elapsedCP - the elapsed time since the last checkpoint in the standard TimeSpan format ([-][d.]hh:mm:ss[.fffffff])
  • elapsedCP:ms - the elapsed time since the last checkpoint in milliseconds
