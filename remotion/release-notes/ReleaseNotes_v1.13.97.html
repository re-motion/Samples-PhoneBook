[RM-3791] Changed namespace and assembly of re-linq from Remotion.Data.Linq to Remotion.Linq

Resolution: Fixed
Status: Closed


Relocated re-linq assemblies frol folder .\Data to .\Relinq

Changed the following namespaces and assembly names:

  • Remotion.Data.Linq -> Remotion.Linq
  • Remotion.Data.Linq.SqlBackend -> Remotion.Linq.SqlBackend

Issue Type: Breaking Change
Resolution: Fixed

CustomAttributeData does not support public properties of type Enum.
http://connect.microsoft.com/VisualStudio/feedback/details/296032/customattributedata-throws-when-attribute-has-a-public-enum-property

[RM-3757] Provide paramless overloads of ObjectFactory.Create

Component/s: Mixins
Issue Type: New Feature
Resolution: Fixed
Status: Closed


When you create an instance via ObjectFactory.Create<T>() resp Create(Type), it is necessary to pass ParamList.Empty. Add parameterless overloads for ObjectFactory.Create<T>() and Create (Type) for convenience.
